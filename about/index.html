About Subtify

Subtify is open-source platform that helps you to find words in subtitles or captions from YouTube videos quickly, and it's 100% free!

Fyi, this project is use API from Jefrydco and Google. Google APIs is not free :( so i hope you support this project to keep this project alive :)
